 Language Translation Tool

A responsive Language Translation Web App built using HTML, CSS, and JavaScript. The application allows users to translate text between multiple languages through API integration while providing a clean, modern, and user-friendly interface.

 Features

* Translate text between multiple languages
* Select source and target languages
* Real-time translated output
* Copy translated text to clipboard
* Text-to-speech functionality
* Responsive and modern UI design
* Easy-to-use interface

Technologies Used

* HTML5
* CSS3
* JavaScript
* Translation API

Project Overview

This project was developed to improve frontend development skills and gain practical experience working with APIs and responsive web application design. The application focuses on usability and accessibility by integrating additional features such as copy functionality and speech synthesis.
